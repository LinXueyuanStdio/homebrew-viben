# Homebrew Tap for Viben

Homebrew formulae for [Viben](https://github.com/LinXueyuanStdio/viben) - Orchestrate AI agent clusters in your local workspace.

## Installation

```bash
brew tap LinXueyuanStdio/viben
brew install viben
```

## One-liner

```bash
brew install LinXueyuanStdio/viben/viben
```

## Other Installation Methods

```bash
# Shell script (macOS/Linux)
curl -fsSL https://github.com/LinXueyuanStdio/viben/releases/latest/download/install.sh | bash

# npx (Node.js 18+)
npx viben

# npm global
npm install -g viben
```

## Requirements

- Node.js 18+
